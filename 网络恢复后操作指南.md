## 🔍 检查网络连接

### 测试 GitHub 连接

```bash
ping github.com
```

或者：

```bash
curl -I https://github.com
```

### 如果连接失败

1. **检查网络设置**
   - 确保网络连接正常
   - 检查防火墙设置
   - 检查代理设置

2. **尝试使用代理**
   - 如果使用 VPN，确保 VPN 连接正常
   - 如果使用代理，配置 Git 代理设置

3. **稍后重试**
   - 等待网络恢复后重试
   - GitHub 服务器偶尔也会繁忙
