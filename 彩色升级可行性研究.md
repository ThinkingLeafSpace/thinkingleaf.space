## 彩色升级可行性研究（马蒂斯 × 禅意）

### 背景与目标
当前站点以米白/炭灰/朱红为主，呈现“禅意留白”的极简风格。你提出将“马蒂斯式大胆色彩”与“禅意留白肌理”叠加，形成更具冲击力的彩色体验，同时保持内容可读性与品牌辨识度。本文件评估可行性，并提供7套高饱和情绪语义配色（类似 LUT 预设）以便快速试色与切换。

### 可行性结论（摘要）
- 技术上：完全可行。基于现有 CSS 变量体系，新增“情绪主题变量”与“类别主题挂载”即可无痛切换，兼容深色模式。
- 设计上：可在“宣纸米白+留白布局”上叠加高饱和块面、边框、图形与渐变，获得马蒂斯式活力；朱红继续作为品牌主强调色或作为其中一套主题的核心。
- 风险可控：通过对比度守则、颜色层级规范与灰度回退策略可降低可读性与喧宾夺主风险。

### 设计方向与原则
- 马蒂斯 × 禅意：
  - 禅意支柱：留白、质感（paper-texture）、低视觉噪声的排版秩序。
  - 马蒂斯支柱：高饱和纯度色块、清晰边缘、反常规但“合情理”的对比搭配。
- 用色策略：
  - 彩色承担“情绪/导航/分区/行动”角色；正文可读性仍以高对比为准（深色文字/浅色底，或反之）。
  - 减少大面积多彩并行；以版块/组件为单位使用“单主题+少量对比强调”。
  - 统一的交互状态色（hover/active/focus）在各主题中按“等情绪位阶”变化，不混乱。

### 风险与对策
- 可读性风险：对比度遵循 WCAG AA（正文≥4.5:1；大字≥3:1）。
- 视觉疲劳：控制高饱和色块的面积与密度，优先用于边框、分隔、徽章、按钮、插图背景。
- 品牌一致性：保留朱红在全站的“重要动作”位阶；其它主题仅在内容域生效。
- 深色模式：提供对应“暗值映射”，避免直接暗化导致脏灰。

### 深色模式研究要点
- 非线性映射：不是简单 HSL 降明度；需为每套主题定义暗色等效（Hue 轻微偏移以维持色相辨识）。
- 反差位阶：深色模式中，Accent 亮度上调 8–12 以维持层级；边框/阴影用“彩色暗影”而非黑灰。
- 纸纹理：暗色模式降低纹理对比与颗粒（opacity 由 0.98 → 0.92）。

### 变量命名与挂载方式（不改动现有基础变量）
建议新增一组主题变量（示例为“激情”主题）：

```css
:root[data-emotion="vitality"] {
  --theme-primary: #E63946;
  --theme-secondary: #FFB703;
  --theme-accent: #2A9D8F;
  --theme-support: #264653;
  --theme-ink: #1D1D1F;
}

@media (prefers-color-scheme: dark) {
  :root[data-emotion="vitality"] {
    --theme-primary: #FF5566;
    --theme-secondary: #FFC44D;
    --theme-accent: #34B6A3;
    --theme-support: #1C2A35;
    --theme-ink: #F5F5F5;
  }
}
```

应用规则（不破坏正文可读性）：
- 按组件位阶使用：
  - 标题/强调线/按钮：var(--theme-primary / --theme-accent)
  - 卡片边框/标签底/分割：var(--theme-secondary)
  - 图标/描边/阴影：var(--theme-support)
  - 正文文本：var(--theme-ink)

DOM 挂载：
- 在 `body` 或 `.main-content` 上设置 `data-emotion="vitality"` 即可切换。
- 类别页/文章依据 FrontMatter 标签决定 `data-emotion` 值（后续脚本可自动注入）。

### 渐进落地计划
1. 主题变量引入（不改现有色）：仅添加 7 套 `data-emotion` 变量，无 UI 变更。
2. 小范围试色：在首页“卡片边框/标签/按钮”三处挂载主题变量，评估冲击力与对比度。
3. 类别级应用：为 `blogs/cabinet/portfolio/newsletter` 四块分别挂载默认主题。
4. 情绪标签驱动：文章按标签自动切主题；提供“试色切换器（像 LUT）”。
5. 深色模式映射：补齐 7 套暗色等效并联调。
6. 性能与可访问性校验：CLS/LCP、对比度、键盘可达性复测。

验收指标（建议）：
- 可读性：正文对比度 AA 通过率≥99%。
- 识别度：A/B 问卷中“版块快速识别”正确率≥85%。
- 参与度：按钮点击率/滚动深度较基线提升 8–12%。
