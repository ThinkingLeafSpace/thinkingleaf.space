---

## 📝 第一步：写文章

### 方式1：使用模板（推荐，1分钟）

1. 打开Obsidian
2. 在 `/Users/qianny/Nutstore Files/Qianny-obsidian/个人网站` 文件夹
3. 按 `Cmd+P`，输入"模板"，选择"插入模板"
4. 修改Front Matter：
   ```yaml
   ---
   title: "你的文章标题"
   date: "2025-11-02"  # 今天日期
   description: "简短描述"
   category:
     - 标签1
     - 标签2
   ---
   ```
5. 写正文内容
6. 保存（Cmd+S）

✅ **耗时：1-2分钟**

---

## 🚀 第二步：一键部署

### 最简单方式：在Obsidian中配置命令

#### 第一次设置（5分钟，之后永久）

1. 安装 Commander 插件（社区插件搜索"Commander"）
2. 设置 → Commander → Add Command
3. 填写：
   - **名称：** `部署博客`
   - **类型：** `Shell commands: Execute shell command`
   - **命令：**
   ```bash
   cd "/Users/qianny/Nutstore Files/ThinkingLeafSpace 20250704com/ThinkingLeafSpace 20250924com" && ./scripts/deploy_blog.sh "{{file.path:absolute}}"
   ```
4. 可选：设置快捷键 `Cmd+Shift+D`
5. 保存

#### 每次部署

1. 打开要部署的文章
2. 按 `Cmd+P`，输入"部署"，回车
3. ✅ 完成！

**耗时：3秒**

---

### 备选方式：终端命令

如果不想装插件：

1. 打开终端（`Cmd+空格`，输入"终端"）
2. 复制粘贴运行：
```bash
cd "/Users/qianny/Nutstore Files/ThinkingLeafSpace 20250704com/ThinkingLeafSpace 20250924com"

./scripts/deploy_blog.sh "/Users/qianny/Nutstore Files/Qianny-obsidian/个人网站/你的文章.md"
```

**提示：** 拖拽文件到终端自动填入路径

**耗时：10秒**
