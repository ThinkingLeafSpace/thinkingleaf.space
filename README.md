## 4. 灵魂配图系统 (Soulful Image Curation)

"灵魂配图SOP" (V2.0) 是我们兼顾"灵魂Vibe"与"性能洁癖"的配图工作流。

### 核心原则

- **灵魂策展**: 100%优先使用自己的"灵魂照片"（工作坊、禅修、旅行中的故事性照片）
- **性能洁癖**: 使用WebP格式，确保图片<150KB，通过Squoosh.app或TinyPNG.com压缩
- **Vibe Coding**: 使用语义化的`<figure>`和`<figcaption>`标签，设置`width`和`height`防止CLS，添加`loading="lazy"`优化性能

### 配图位置

- **位置A（灵魂定调图）**: 文章开头，第一个章节标题之前
- **位置B（灵魂呼吸点）**: 关键章节之后，正文之前
- **位置C（灵魂印记）**: 文章结尾，后记部分

### 相关文档

- `灵魂配图SOP.md` - 完整的配图流程文档
- `灵魂配图代码模板.html` - 可直接复制的代码模板
- CSS样式已内置在 `css/styles.css` 中，自动应用淡入动画、优雅阴影等效果
