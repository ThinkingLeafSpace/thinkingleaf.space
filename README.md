# 筑居思：Remake 排列组合引擎 (V1.0)

"叶芽之下，别有根系。" 这是一个探索"有限选择"与"无限组合"的灵魂游戏。

## 1. 它是什么？(What)

"Remake Engine"是一个轻量级的"灵感生成器"。

它基于创始人（你）的个人"灵感库"（如 设计、穿搭、食谱），通过"老虎机"式的随机组合，为"同路人"创造"啊哈！"的惊喜瞬间。

**核心使命：** 帮助用户打破"消费主义"的"现成状态"，回归"上手状态"的创造乐趣。

## 2. 灵魂问题 (Why)

- 我们厌倦了"最优解"：算法总在推荐"标准答案"。
- 我们渴望"无用之用"：我们想"玩弄"时间，而不是"管理"时间。
- 我们相信"限制就是自由"：有限的"牌组"能激发无限的"创意"。

## 3. 如何"把玩"？(How) - 开发者

这是一个"静态 MVP"(V1.0) 架构，追求极致的"性能洁癖"。

### 克隆 (Clone)

```bash
git clone [你的仓库URL]
```

### 数据源 (Data)

所有"灵感牌组"都储存在 `public/data/soul-deck.json`。

> 作为你的技术导师，我建议我们 V1.0 先用 JSON，V2.0 再上 Headless CMS

### 启动 (Run)

```bash
npm install
npm run dev
```
